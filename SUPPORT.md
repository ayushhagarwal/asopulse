# Support

- Use [GitHub Discussions](https://github.com/ayushhagarwal/asopulse/discussions) for setup questions, troubleshooting, and product ideas.
- Use [Issues](https://github.com/ayushhagarwal/asopulse/issues) for reproducible bugs.
- Use private vulnerability reporting for security problems as described in [SECURITY.md](SECURITY.md).

Community support is best effort. Include the ASOpulse version, deployment model, relevant sanitized logs, and what you already tried. Never post credentials, complete environment files, database exports, or private user data.
